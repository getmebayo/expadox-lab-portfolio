# Evidence Index

This folder is intended for evidence demonstrating that the monitoring environment operated as designed.

Suitable evidence includes:

- Wazuh dashboard screenshots
- Suricata alerts
- Authentication logs
- Hydra execution results
- Nmap scan results
- Email alert screenshots
- Custom rule files
- Incident-response records
- Agent connection status
- Tailscale connectivity evidence

Do not include passwords, API keys, private IP information that should remain confidential, or production credentials.
